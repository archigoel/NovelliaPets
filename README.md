Novellia Pets 🐾
Mobile Medical Records for your Furry Friends
Novellia Pets is a React Native MVP designed to help pet owners manage their pets' health journeys. It features a modern UI, a localized Express backend, and persistent user sessions.

# Getting Started
Follow these steps to get the app running on your machine.

### 1. Prerequisites
Node.js (v18 or higher recommended)

Expo Go app installed on your physical device or an Emulator.

### 2. Installation
You must install dependencies in both the root and the server directory:

#### Install root dependencies
npm install

#### Install backend dependencies
cd server && npm install && cd ..

### 3. Run the App
Start both the Express backend and the Expo bundler with a single command:

npm run dev
